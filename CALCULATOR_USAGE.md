# 计算器使用指南

## 🚀 启动计算器

有三种方式启动交互式计算器：

### 方式 1: 使用 npm 脚本
```bash
npm start
# 或
npm run calc
```

### 方式 2: 直接运行
```bash
node cli.js
```

### 方式 3: 作为可执行文件（如果已添加到 PATH）
```bash
./cli.js
```

## 📖 支持的运算

### 基本运算
- **加法**: `5 + 3` → 8
- **减法**: `10 - 4` → 6
- **乘法**: `6 * 7` → 42
- **除法**: `20 / 4` → 5

### 高级运算
- **取模（余数）**: `10 % 3` → 1
- **幂运算**: `2 ^ 8` → 256
- **平方根**: `sqrt(16)` 或 `√(25)` → 4 或 5

## 🎮 命令

- `help` - 显示帮助信息
- `clear` - 清除屏幕
- `exit` 或 `quit` - 退出计算器
- `Ctrl+C` - 快速退出

## 💡 使用示例

```
计算 > 5 + 3
结果: 8

计算 > 2 ^ 10
结果: 1024

计算 > sqrt(144)
结果: 12

计算 > 17 % 5
结果: 2
```

## 🛠️ 作为模块使用

如果您想在自己的代码中使用计算器函数：

```javascript
const calculator = require('./src/calculator.js');

console.log(calculator.add(5, 3));        // 8
console.log(calculator.power(2, 8));      // 256
console.log(calculator.squareRoot(16));   // 4
```

## ✅ 运行测试

```bash
npm test
```

## 📝 注意事项

- 所有运算符两边需要有数字
- 支持小数运算：`5.5 + 2.3`
- 除数和取模除数不能为0
- 平方根不能计算负数

## 🖥️ 图形化界面

### 启动图形化计算器

```bash
npm run web
# 或
npm run gui
```

然后在浏览器中打开：**http://localhost:3000**

### ✨ 功能特点

- 🎨 **漂亮的紫色渐变主题**
- 🖱️ **鼠标点击操作** - 点击按钮进行计算
- ⌨️ **键盘支持** - 使用键盘输入数字和运算符
- 📊 **计算历史** - 自动保存最近10条计算记录
- 📱 **响应式设计** - 支持手机和平板

### 🎮 按钮说明

**基本操作：**
- `C` - 清除所有
- `CE` - 清除当前输入
- `⌫` - 退格删除
- `±` - 切换正负号

**基本运算：**
- `+` `-` `×` `÷` - 加减乘除
- `.` - 小数点

**高级运算：**
- `MOD` - 取模（余数）
- `x^y` - 幂运算
- `√` - 平方根

### ⌨️ 键盘快捷键

- `0-9` - 输入数字
- `+ - * /` - 运算符
- `Enter` 或 `=` - 计算结果
- `Esc` - 清除显示
- `Backspace` - 退格

### 💡 使用示例

1. 点击数字按钮输入数字
2. 点击运算符按钮
3. 输入第二个数字
4. 点击 `=` 查看结果
5. 查看底部的计算历史

### 🌐 访问地址

服务器启动后，在浏览器访问：
- **本地访问**: http://localhost:3000
- **手机访问**: http://YOUR_IP:3000 (在同一WiFi下)
